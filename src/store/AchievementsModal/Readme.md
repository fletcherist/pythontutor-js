
### API

Push an achievement.
```javascript
window._achievementsController.achievements.push()
```

@achievements — an array of achievements
ex.
`
		{
      "description": "решено 2 задачи",
      "name": "solved2",
      "img": "/static/images/achievements/new_solved2.png"
    }
`


Open modal window.
```javascript
window._achievementsController.openModal()
```

Close modal window.
```javascript
window._achievementsController.openModal()
```
